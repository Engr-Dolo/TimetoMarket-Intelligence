# TimetoMarket (Layer 1 — Consumer Agri-Marketplace)

USSD, WhatsApp, and Orange Money–enabled marketplace connecting smallholder farmers directly to buyers.

## Planned modules

- `ussd-gateway/` — USSD session handling for farmer access
- `whatsapp-bot/` — WhatsApp Business API integration
- `ai-listing-engine/` — voice-to-listing (NLP), price prediction, quality grading (computer vision), logistics matching
- `payments/` — Orange Money escrow integration

## Status

Design stage. Folder structure reflects planned module boundaries; implementation to follow during the program.
